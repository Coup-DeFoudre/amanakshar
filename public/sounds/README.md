# Ambient Sound Files

This directory should contain the following ambient sound files for the reading experience:

## Required Files

1. **tabla-ambient.mp3** - Soft tabla rhythms (2-5 minutes, loopable)
2. **flute-ambient.mp3** - Peaceful bansuri/flute melody (2-5 minutes, loopable)
3. **sitar-ambient.mp3** - Gentle sitar background (2-5 minutes, loopable)
4. **rain-ambient.mp3** - Soft rain sounds (2-5 minutes, loopable)

## Recommended Sources

You can find royalty-free Indian classical ambient music from:

- [Pixabay](https://pixabay.com/music/search/indian%20classical/)
- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/)
- [Bensound](https://www.bensound.com/)

## File Requirements

- **Format**: MP3 (for best browser compatibility)
- **Duration**: 2-5 minutes (will loop automatically)
- **Quality**: 128-192 kbps (balance of quality and file size)
- **Volume**: Normalize to similar levels across tracks
- **Looping**: Ensure tracks loop seamlessly (fade in/out at edges)

## Important

Make sure you have the appropriate license for any audio files you use.
The ambient sound feature will gracefully handle missing files.
